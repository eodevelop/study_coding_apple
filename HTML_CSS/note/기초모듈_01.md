## HTML 기본 태그로 글 작성해보기

- HTML은 <태그>로 작성한다.
- `<p>` 태그는 paragraph의 약자로 문단을 나타낸다.
- 태그의 내부에 들어가는 것은 속성 이라고 한다.
  - 일부 태그는 이런 속성을 기입이 가능하다
- `<ui>` 태그는 unordered list의 약자로 순서가 없는 목록을 나타낸다.
- `<li>` 태그는 list item의 약자로 목록의 항목을 나타낸다.
